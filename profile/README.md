# 척척학사

척착학사는 수원대학교 학생의 졸업 요건을 분석과 학사 관리를 도와주는 서비스입니다.

## 주요 저장소

| Repository | 역할 |
| --- | --- |
| [`cchaksa-app`](https://github.com/cchaksa/cchaksa-app) | 모바일 앱 |
| [`cchaksa-backend`](https://github.com/cchaksa/cchaksa-backend) | 백엔드 API |
| [`cchaksa-be-terraform`](https://github.com/cchaksa/cchaksa-be-terraform) | 백엔드 인프라 코드 |
| [`cchaksa-scraper`](https://github.com/cchaksa/suwon-scraper-v1) | 포털 스크래퍼 |
